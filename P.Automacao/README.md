# 📂 Sistema de Automação de Ficheiros

Projeto em Python para automação de tarefas:

## 🚀 Funcionalidades
- Organizar ficheiros por categoria
- Detectar ficheiros duplicados (hash sha256)
- Mover ou apagar duplicados
- Criar backups
- Gerar relatórios em CSV
- Configuração dinâmica via config.json

## ⚙️ Tecnologias
- Python
- os / shutil / hashlib
- JSON (configuração)
- CSV
- time

## ▶️ Como usar
1. Editar `config.json`
2. Executar:
```bash
python main.py
